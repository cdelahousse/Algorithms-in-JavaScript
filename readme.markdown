	http://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle

*	insertionSort.js

* KnuthMorrisPratt-StringMatching.js

* quickSort.js

*	selectionSort.js

Everything is stored in the `sort` object;

###USAGE:###

	JS>sort.*sortType*(collection);

###EXAMPLE:###

	JS>sort.insertionSort([6,7,3,4,9,1]);
